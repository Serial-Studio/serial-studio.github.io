Visualize your data.

Serial Studio allows you to easily display, process & export data from your embedded projects. The application is able to interact with serial ports, network sockets & MQTT brokers.

 Install    Source code
"We like Serial Studio’s ease of use and adaptability, and we’ll likely use it for our own projects in the future."

Multiple widgets

Visualize your data with FFT charts, multiple data plots, logarithmic plots, gauges, bar widgets and more.

Export your data

Serial Studio can generate CSV files in real-time from the data received from your embedded device.

Cross-platform

This software runs on Windows, macOS and GNU/Linux. You can port it to other platforms if needed.

Open Source

Since this software is MIT licensed, you can use it commercially!

IoT Ready.

Serial Studio can also act as a simple MQTT client. This allows you to use Serial Studio to receive and display data across multiple computers anywhere in the world.


Read more about MQTT.

Support for integrity checks.

Serial Studio has support for CRC-8, CRC-16 and CRC-32 checksums. For more information, check this issue.

A project for everyone.

Everyone is welcome to use, modify and improve the project.


Even if you don't know how to code, you can contribute to Serial Studio by reporting bugs, providing ideas, translating it in your native language or by spreading the word with your colleagues and friends.
