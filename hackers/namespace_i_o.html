◆ ADD_ESCAPE_SEQUENCES()

static QString IO::ADD_ESCAPE_SEQUENCES ( const QString &  str)
static

Adds support for C escape sequences to the given str. When user inputs "\n" in a textbox, Qt automatically converts that string to "\\n". For our purposes, we need to convert "\\n" back to "\n", and so on with the rest of the escape sequences supported by C.
